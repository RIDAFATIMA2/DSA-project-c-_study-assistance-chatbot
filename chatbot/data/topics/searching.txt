Searching Algorithms

Definition:
Searching algorithms find an element in a collection. Common algorithms: linear search, binary search.

Pseudocode: Linear Search
function linearSearch(arr, key):
    for i = 0 to n-1:
        if arr[i] == key: return i
    return -1
Pseudocode: Binary Search (iterative)
function binarySearch(arr, key):
    low = 0; high = n-1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == key: return mid
        if arr[mid] < key: low = mid + 1
        else: high = mid - 1
    return -1

Example:
arr = [1,3,5,7]; binarySearch(arr,5) -> index 2

FAQ:
Q: When use binary search?
A: When the array (or list supporting random access) is sorted.

Q: Binary search on linked list?
A: Not efficient ,linked lists don't support O(1) random access; prefer other approaches or convert to array.
